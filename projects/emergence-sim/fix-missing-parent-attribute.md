Look for AttributeError messages indicating which attribute is missing\n   - Check the parent class's `__init__` method to see what attributes it sets\n\n2. **Find where to add the initialization**\n   - Add the missing attribute initialization in the subclass's `__init__` method\n   - Place it after setting any subclass-specific attributes but before they're used\n\n3. **Copy the initialization exactly**\n   - Copy the attribute initialization line(s) from the parent class\n   - Ensure you have access to any required constants or imported classes\n\n4. **Test the fix**\n   - Run the code to verify the AttributeError is resolved\n   - Ensure the subclass still behaves as intended with its customizations\n\n## Example\n\n**Parent class (sim.py):**\n```python\nclass Simulation:\n    def __init__(self, num_agents=50, use_predator=False, use_seasons=False):\n        # ... other initialization\n        self._grid = SpatialGrid(WIDTH, HEIGHT, cell_size=13.0)\n        self.use_seasons = use_seasons\n        # ... more initialization\n```\n\n**Subclass with missing attribute (run_dialect_test.py):**\n```python\nclass DialectSimulation(Simulation):\n    def __init__(self, num_agents_per_side=25):\n        # Don't call super().__init__ — we'll set up manually\n        self.tick = 0\n        self.agents = []\n        # ... other custom initialization\n        self.use_seasons = False\n        # MISSING: self._grid initialization\n```\n\n**Fixed version:**\n```python\nclass DialectSimulation(Simulation):\n    def __init__(self, num_agents_per_side=25):\n        # Don't call super().__init__ — we'll set up manually\n        self.tick = 0\n        self.agents = []\n        # ... other custom initialization\n        self.use_seasons = False\n        self._grid = SpatialGrid(WIDTH, HEIGHT, cell_size=13.0)  # ← ADDED\n        # ... rest of initialization\n```\n\n## Verification\n- The AttributeError for the missing attribute should be resolved\n- The subclass should retain its custom behavior\n- All methods that depend on the missing attribute should work correctly\n\n## Related Skills\n- systematic-debugging: For identifying and fixing attribute errors\n- plan: For planning complex debugging sessions\n\n## Limitations\n- Only works when you know exactly what attribute is missing and how to initialize it\n- Doesn't address cases where the parent initialization has side effects beyond attribute setting\n- For complex inheritance hierarchies, you may need to initialize multiple parent attributes\n
