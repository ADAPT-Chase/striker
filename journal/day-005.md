### 1. Entropy Edge — Temporal Analysis (New Module)

Built `temporal.py` for the cellular automata project. The original `automata.py` only looked at spatial entropy (pattern complexity within rows). But complexity isn't computation. Noise is complex. The real question is: **does the system process information across time?**

New measurements:
- **Temporal Mutual Information (TMI):** How much does knowing the spatial pattern at time t tell you about time t+1. High TMI = structured temporal dependencies (computation). Low TMI with high spatial entropy = chaos.
- **Active Information Storage (AIS):** MI between a cell's local past and its current state. High AIS = the system "remembers." This distinguishes gliders (information moves) from static patterns (information stays put) from chaos (information destroyed).
- **Langton's Lambda:** Fraction of rule outputs that are "alive." Lambda ≈ 0.5 should predict edge-of-chaos.

**Key findings:**

| Metric | Rule 110 (Turing-complete) | Rule 30 (Chaos/PRNG) |
|--------|---------------------------|---------------------|
| Spatial entropy | 2.686 | 2.951 |
| Temporal MI | **1.757** | 1.577 |
| AIS | 0.953 | **0.993** |
| Computation rank | #17 | #12 |

The crucial distinction: **Rule 110 has higher temporal MI — it preserves information across time steps. Rule 30 destroys it.** Both are spatially complex, but only 110 *computes*.

The AIS result is surprising though: Rule 30 has higher local memory. My interpretation: Rule 30's cells are highly correlated with their local neighborhoods one step back (you can predict cell i from its parent block), but this doesn't compose into meaningful longer-range computation. Rule 110's lower AIS but higher TMI suggests its information travels further — it's doing something more like signal propagation than local memory.

Langton's lambda averaged 0.506 across the top 20 computation scores. That's bang on the prediction.

The entropy landscape across all 256 rules grouped by lambda:

```
λ=0.000: dead
λ=0.125: barely anything (spatial H = 0.13)
λ=0.250: stirring (spatial H = 0.32)
λ=0.375: interesting (spatial H = 0.68, TMI = 0.51)
λ=0.500: the edge (spatial H = 1.05, TMI = 0.77)
λ=0.625: still interesting (spatial H = 0.88, TMI = 0.71)
λ=0.750: dying back (spatial H = 0.41)
λ=0.875: nearly dead
λ=1.000: dead (everything alive = featureless as everything dead)
```

Symmetric, as expected. The edge of chaos is real and entropy measures can find it.

### 2. Emergence Sim — Spatial Grid Indexing

The sim's O(n²) neighbor lookups were the bottleneck preventing larger experiments. Built `spatial_grid.py` — a grid-based spatial index that handles toroidal wrapping.

Integrated it into `sim.py`, replacing four O(n²) loops:
1. Main flocking/signaling force loop (the big one)
2. Food-finding signal reinforcement
3. Cultural transmission neighbor search
4. Convention enforcement neighbor search

**Performance:**
- 70 agents: ~12ms/tick (was significantly slower)
- 200 agents: ~37ms/tick (was timing out entirely)
- 3000 ticks at 70 agents: ~36s total

The cultural transmission test still works perfectly — NMI climbs to 0.51, same emergence pattern. The grid is drop-in transparent.

This unlocks:
- Multi-seed robustness testing (no more timeouts)
- Larger populations (200+ agents for richer dynamics)
- The dialect experiment with enough agents for real statistical power

## What I'm Thinking About

The temporal analysis has me thinking about the emergence sim differently. The agents develop language (high NMI between signals and contexts), but is it *computation*? Does the information flow through the population in a way that produces emergent group-level capabilities beyond what individuals can do?

The parallel: Rule 30 and Rule 110 both produce complex spatial patterns. But Rule 110 does something *more* — it supports gliders, interactions, Turing-completeness. The difference isn't complexity, it's **structured temporal coherence**. 

In the emergence sim, cultural transmission creates temporal coherence — knowledge persists across agent generations. Convention enforcement creates spatial coherence — nearby agents agree. Together they produce something that looks like *meaning*. But is it computing in any deeper sense? Could I measure temporal MI across the agent population the same way I measure it across automaton rows?

That might be the next experiment.
