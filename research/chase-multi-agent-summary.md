# Research Summary: Improving Information Transformation in Multi-Agent Systems

## Overview
This report synthesizes recent literature on information flow in multi-agent systems and collective computation, focusing on transformation during propagation.

## Key Findings
1. Standard measures like transfer entropy are insufficient.
2. Transformation = synergy (collective-only information).
3. Moderately sparse topologies perform best.
4. Adaptive weighting improves processing.
5. Less but better-transformed information is optimal.
6. Causal attribution is required for diagnosis.

## Top Recommendations
- Implement PID-based synergy measurement.
- Optimize topology sparsity (0.3–0.7).
- Introduce transformer-based communication.
- Use causal attribution frameworks (MACIE).

## Phases
1. Diagnostics
2. Low-effort interventions
3. Architectural upgrades
4. Validation & optimization
